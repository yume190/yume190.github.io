術語

Annotation: 關於 時間(timestamp) 服務(service) host 的字串資料

  • Annotation
    • cs - Client Start.
    • sr - Server Receive.
    • ss - Server Send.
    • cr - Client Recieve.
  • BinaryAnnotation : 沒有時間元素，他們代表額外的 RPC 資訊

Span: 代表一個特定的方法呼叫，由一組(set) annotation 組成，並且有 名稱 及 ID

Trace: 紀錄同一個 Request 的一組 span
